## 📌 项目背景

随着中国轨道交通事业的高速发展，越来越多的公众对轨道交通技术、历史与运维文化产生了浓厚的兴趣。但传统线下科普场馆受限于开放时间、地理位置与接待容量，难以满足大众随时随地、系统学习的需求；同时，行业专业知识分散在各类资料中，普通爱好者、中小学生缺乏轻量化、结构化的学习渠道，传统科普的单向传播模式也难以调动用户的参与热情。

为解决**传播范围有限、互动性不足、内容更新滞后、单向传播参与度低**的核心痛点，我们打造了这个数字化轨道交通主题博物馆，以知识普惠为目标，搭建一个永久在线、免费开放、可持续共建的科普平台，让轨道交通知识突破物理限制，触达更多人群。

## 📋 项目简介

本项目是一个**无后端、纯静态、零依赖**的轨道交通科普平台，主要面向在校师生、轨道交通爱好者、中小学生及广大公众开放。

通过结构化的展厅、深度专题与共建体系，项目实现了轨道交通知识的高效传播与互动式学习，大幅提升了科普内容的可访问性与学习效率，让任何人都能随时随地、免费获取专业的轨道交通知识。

## ✨ 核心功能

- **多主题核心展厅**：涵盖动力机械厅、历史演进厅等专业模块，系统展示轨道交通技术与发展历程

- **深度特展专题**：推出「守夜人的光轨」等一线运维实录专题，还原轨道交通幕后工作场景

- **共建贡献体系**：搭建荣誉馆员任务板，支持爱好者参与内容共建，打造全民科普共创平台

- **全平台适配**：支持电脑、手机、平板全终端访问，随时随地可浏览学习

- **零门槛使用**：无需注册、无需安装、无需数据库，打开即可浏览所有内容

- **便捷导航检索**：清晰的内容目录与导航，快速定位你感兴趣的科普内容

## 🚀 快速开始

本项目为纯静态项目，无需配置复杂环境，开箱即用。

### 本地运行

1. **克隆项目仓库**

   ```bash
   git clone [项目仓库地址]
   cd 数字化轨道交通主题博物馆
   ```

2. **启动静态服务（二选一）**

   - 方式 1：使用 VS Code 插件
     安装 `Live Server` 插件，直接右键打开项目根目录的任意 HTML/MD 文件即可自动启动服务。

   - 方式 2：使用命令行工具
     若已安装 Node.js，可通过 `http-server` 快速启动：

       ```bash
     npm install -g http-server
     http-server -p 8080
       ```

3. **访问项目**
   浏览器打开 `http://localhost:8080` 即可本地预览完整项目。

## 📖 使用指南

### 普通用户

1. 打开线上 / 本地项目地址，无需注册登录

2. 通过导航栏选择「核心展厅」或「特展」进入对应模块

3. 浏览阅读科普内容，可通过页面目录快速跳转章节

### 项目维护者 / 贡献者

1. 项目权限通过 GitHub 仓库权限管理，维护者拥有仓库读写权限

2. 内容更新：直接修改仓库内对应目录的 Markdown 文件

3. 提交更新后，可通过 GitHub Actions 自动同步到线上部署地址

## ⚠️ 注意事项

1. 本项目为纯静态网站，无后端服务、无数据库、无用户系统，所有内容通过文件直接管理

2. 图片资源建议使用相对路径，保证本地运行与线上部署均可正常加载

3. 本地启动服务时若端口被占用，可通过 `-p` 参数自定义端口（如 `http-server -p 8081`）

4. 建议配置 GitHub Actions 自动部署流程，实现代码提交后自动同步到线上，提升内容更新效率
