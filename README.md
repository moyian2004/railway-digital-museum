# 🚇 轨道交通数字博物馆
Railway Digital Museum

基于开源协作与人工智能赋能的数字化轨道交通科普平台

## 项目介绍
本项目面向交通强国、数字中国建设需求，聚焦轨道交通科普内容分散、形式单一、互动性不足等问题，打造集数字展陈、知识科普、AI问答、3D/VR互动于一体的线上数字博物馆。
面向公众科普、新工科人才培养、轨道交通专业学习与院校特色展示。

## 已完成展厅
### 核心技术展厅
- 转向架：列车的“超级滑板”，包含结构、原理、转弯机制、高铁与地铁应用、智能监控

## 核心功能
- 数字化展厅：图文 + 3D 模型展示轨道交通核心技术
- AI 智能问答：自动解答轨道交通专业问题
- 开源协作：支持师生共同完善内容、提交改进建议
- 3D/VR 互动：转向架拆解、列车运行模拟等沉浸式体验
- 多端适配：手机、平板、电脑均可访问

## 技术栈
- 前端：Vue / HTML + CSS + JavaScript
- AI：大模型问答、知识整理
- 数字展示：3D 模型、VR 场景、交互式图解
- 协作：GitHub 开源协作
